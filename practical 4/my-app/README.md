Facebook Clone

A simple Facebook clone built with Next.js, React, TypeScript, and Tailwind CSS. It supports user authentication, posting, and a chronological social feed.

--Features
User signup & login
Create and view posts
Chronological feed
Responsive design
Data stored in localStorage

--Tech Stack
Next.js, React, TypeScript
Tailwind CSS
React Icons

--Structure
layout.tsx: App layout and fonts
page.tsx: Main logic (auth, posts, UI)
globals.css: Styling and theme

--How It Works
Users can register/login
Auth state and data saved in localStorage
Posts are created and shown in reverse order

--Run
npm install
npm run dev

--Limitations
No backend (localStorage only)
No real-time updates or advanced features

--Future Improvements
Backend + database
Profiles, likes, comments
Better security
